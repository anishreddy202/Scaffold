Wireless Network Services (WNS) Overview

At the heart of every wireless M2M application are the wireless devices that produce asset data that is later routed to the back-end application servers – such as meter readings, sensor data, readings from medical equipment, and so forth. To improve operational efficiency and resources, M2M applications can automate the provisioning, monitoring, and control of wireless M2M devices by leveraging Verizon's WNS.

Wireless Network Services (WNS) provides APIs that allow you to integrate device management functions into enterprise management systems and custom applications. WNS consists of a set of standards-compliant SOAP/XML Web Services that offer you a single, portable, and easy-to-use interface to the Verizon Wireless network, reducing the complexity of developing M2M applications. WNS shields developers from the underlying complexity of M2M applications. For example, to change the rate plan for a deployed device, an M2M application simply calls a single WNS web service API, Change Device Rate Plan, and provides two parameters: a device ID and a desired price plan.
